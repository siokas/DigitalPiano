DigitalPiano

Play music from Arduino using the accelerometer sensor of an Android device

View project onGitHub

Welcome to DigitalPiano app

The DigitalPiano app is a combination of Android-Processing-Arduino applications which is playing piano in the speaker connected to the Arduino. The Android app is taking values from the accelerometer sensor and sends them to the Processing app via the WiFi network using Sockets. The Processing app is a simple server which forwards to the Arduino (using the Serial connection) the values which are taken from the Android app. The Arduino app takes the values from the Serial connection and depending to these values sends the corresponding note (pitch) to the connected speaker.

Android app

The Android app is taking values from the accelerometer sensor and sends them to the Processing app via the WiFi network using Sockets. There is a "Start Accelerator" button which starts the sensor and an input field to enter the ip of the server. If the server's ip is correct, up and running and the sensor is taking values of the x axis, the app makes a connection with the server and sends the x values in an absolute format (because if you tilt the device to the right, the x axis takes negative values).

SensorManager sensorManager = (SensorManager) getSystemService(SENSOR_SERVICE);

List<Sensor> sensors = sensorManager.getSensorList(Sensor.TYPE_ACCELEROMETER);

Processing app

The Processing app is a simple server which forwards to the Arduino (using the Serial connection) the values. It creates a small window and starts the server socket waiting for incoming connections. If there is a connection, the app prints the ip of the client and forwards the incoming message to the Serial connection (in which the Arduino is connected).

Arduino app

The Arduino app takes the values from the Serial connection and depending to these values sends the corresponding note (pitch) to the connected speaker. The incoming values are between 0 and 9 and the Arduino app plays different note depending on theses values.

Arduino Circuit

To play music with your Arduino you need to have: an Arduino (any type), a breadboard (not necessary), a speaker and some breadboard wires. You have to connect the one cable of speaker to an Arduino pin and the other to the Ground.

Troubleshooting

In the Processing code you have to change the Serial port list to the one which is connects your Computer with the Arduino. Change the mySerialPortList

int mySerialPortList = 0;

String portName = Serial.list()[mySerialPortList];

In the Arduino code you have to change the pin to the one which you have connected the speaker in the Arduino.

int pin = 12;

Step by Step

  1. Download the Android porject, find the .apk file and install it on your device
  2. Download the Processing project and run the application file
  3. Download the Arduino project
  4. Connect the speaker to the Arduino (there are information about it above)
  5. Connect the Arduino to your computer via Serial camble and upload the sketch
  6. Launch the Android app on your smartphone or tablet
  7. Enter the server's ip (it is the ip of your computer)
  8. Tilt your devise to the left or to the right in order to play music
  9. Have fun!!!
